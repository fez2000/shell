# SHELL

Mathematical shell for the processing of mass data supporting linux commands. the project is form of the commands and the shell.
To have the project
```
$ git clone https://github.com/clotair/shell.git
$ cd shell
```
## Build shell
```
$ cd shell
$ make
```
### Launch shell `./shell`

### Build some command
```
$ cd commandName
$ make
```
### Launch command `./commandName`
